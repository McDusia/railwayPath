
1. Create new empty database LosAngelesParcels.
2. Copy Parcel table to new Database LosAngelesParcels from LosAngelesCounty database.

Use Microsoft SQL Server Management Studio 2016 
Right click on  LosAngelesParcels database name in Object Explorer
Tasks -> Import Data -> Next -> specify the Data Source (for example SQL Server Native Client 11.0), Server Name,
choose Authentication and Database (LosAngelesCounty) and click Next, choose the Destination: (for example SQL Server Native Client 11.0), Server Name,
choose Authentication and Database (LosAngelesParcels), choose Copy data from one or more tables or views,
choose Parcel Table in left pane (Source: [Server Name]), click Next, choose Run imediately and Finish.
Then it started to copy Parcel table to destination Database LosAngelesParcels.

Alternatively: you can use different solution to copy data, for example save table results in csv file and then load it to new table in LosAngelesParcels database.

Check if you copy data properly. Run (select top 10 * from Parcel) on database LosAngelesParcels.

Note: All scripts you will find in SQL scripts directory.

3. Add indexes and constraints to Parcel table. Run script DDL_PK_Constraints_Indexes.sql.

4. Run DDL_Create_Mapping_Tables.sql script. It will create schema for tables to Map all not integer values from Parcel table.

5. Run all scripts, which create stored procedures needed to run program properly. 
Run:
sp_GetColumnsWithNullValues.sql
sp_GetRowsWithNullValues.sql
sp_UpdateNullValues.sql

	EXEC GetRowsWithNullValues
